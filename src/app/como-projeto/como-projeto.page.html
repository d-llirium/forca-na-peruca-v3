criando o projeto instalações no macbook 1. instalar o VScode 2. instalar o NodeJS ainda não acabou não... @ terminal 1. criar pasta
$ mkdir nomeDaPasta 2. entrar na pasta
$ cd nomeDaPasta 3. instalar o ionic na pasta como admin
$ sudo npm install -g @ionic/cli 4. iniciar um projeto ionic
$ ionic start projeto-app
> angular
> blank 5. iniciar um repositório local
$ git init
* para isso é necessário ter o git instalado na máquina se fosse só isso tava bom né?! @ github 1. ter uma conta no github que esteja conectada ao repositório local do seu computador 2. entrar no link seus repositórios 3. click em novo 4. nome do repositório projeto 5. deixe o repositório público
* não inicialize o repositório README, .gitignore ou license vai seguindo o rítmo.... @ terminal 1. na pasta
$ cd projeto-app 2. adicionar os aquivos do projeto ionic, que foi criado, ao repositório local
$ git add . 3. fazer o primeiro commit, produzindo o primeiro registro de versionamento local
$ git commit -m "commit inicial" 4. adicionar o repositório remoto
$ git remote add origin https://github.com/usuario/repositorio.git
* nesse caso substitua com o https do seu git 5. crie localmente e sinalize que a branch main local é a principal
$ git branch -M main
6. empurre o commit inicial do seu repositório local para o git remoto
$ git push -u origin main eita lasquera!!!! @ VScode 1. abrir a pasta do projeto
[cmnd] + [O] 2. abrir o terminal do VScode
[ctrl] + [shft] + [`] 3. rodar o app no navegador
